# **App Name**: JobTracker Pro

## Core Features:

- Authentication: Secure user authentication using Firebase Auth with email/password and Google Sign-In.
- Job Application Tracking: Add, update, and delete job applications with details such as company, role, status, and dates.
- Real-Time Updates: Utilize Firestore onSnapshot for real-time updates of job application data.
- Dashboard Summary: Display summary cards for total applications, interviews, and offers, along with filters for status, region, and date range.
- Interview Reminders: Set and manage reminders for upcoming interviews, integrating with user's calendar if desired.
- CSV Import/Export: Import and export job application data in CSV format for bulk management.
- AI-Powered Insights and Recommendations: Analyze job application data to provide insights such as success rates, optimal application timing, and personalized recommendations for improving the application strategy, leveraging AI tool to assess the relevance of incorporating the data.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to convey professionalism and sophistication.
- Background color: Light purple (#F2EEFA), a very desaturated version of the primary color.
- Accent color: Dark blue (#3849AA), a saturated hue analogous to the primary color, offering contrast.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text. Space Grotesk offers a techy, scientific feel that suits a modern application tracker.
- Use clean and professional icons from a library like FontAwesome or Material Icons to represent application status, actions, and data points.
- Implement a responsive grid layout that adapts to different screen sizes, ensuring optimal viewing on both mobile and desktop devices.
- Incorporate subtle animations and transitions to enhance user experience, such as smooth loading indicators, form feedback, and chart animations.
